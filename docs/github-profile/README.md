### Ishrith Gowda

UC Berkeley EECS | Medical AI Researcher

I develop deep learning methods for brain MRI harmonization -- standardizing multi-institutional neuroimaging data so that scanner-induced domain shifts do not confound downstream clinical analysis. My work combines generative adversarial networks with attention mechanisms and federated learning to enable privacy-preserving, multi-site medical image translation.

**Research Interests:** Medical Image Analysis, Brain MRI Harmonization, Domain Adaptation, Federated Learning, Generative Models
